# Fluxa Cardápio V9

Sistema de cardápio digital multi-tenant da Fluxa. Desenvolvido com React, TypeScript e Vite.

## 🚀 Tecnologias
- **React 19**
- **TypeScript**
- **Vite**
- **React Router Dom 7**
- **Supabase** (Autenticação e Banco de Dados)
- **Tailwind CSS / Vanilla CSS**

## 📂 Estrutura do Projeto
- `src/components`: Componentes reutilizáveis (Navbar, Sidebar, etc.)
- `src/pages`: Telas principais da aplicação
- `src/context`: Gerenciamento de estado (Carrinho, Autenticação, Empresa)
- `src/hooks`: Custom hooks
- `src/lib`: Configurações de bibliotecas externas (Supabase)
- `src/assets`: Imagens e ícones estáticos

## 🛠️ Como rodar o projeto
1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

## 🌐 Deploy
O projeto está configurado para deploy automático via Vercel (`vercel.json`).
